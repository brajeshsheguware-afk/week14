Credit Card Fraud Detection System
Project Overview

This project analyzes credit card transactions to identify fraudulent activities using Exploratory Data Analysis (EDA), Linear Regression, and Dashboard Visualization. It helps understand transaction patterns and supports fraud detection.

Objectives
Load and analyze the credit card fraud dataset.
Clean and preprocess the data.
Perform statistical and fraud behavior analysis.
Visualize transaction patterns using charts and graphs.
Build a Linear Regression model.
Create an interactive dashboard using Plotly.
Generate insights for fraud prevention.
Dataset

Source: Kaggle Credit Card Fraud Detection Dataset

Features Used
Time – Time elapsed between transactions
Amount – Transaction amount
V1 – V28 – Anonymized transaction features
Class – Target variable (0 = Normal, 1 = Fraud)

Conclusion

The project helps understand fraud patterns, transaction behavior, and risk levels. The visualizations and dashboard provide useful insights that can support better fraud detection and decision-making.
